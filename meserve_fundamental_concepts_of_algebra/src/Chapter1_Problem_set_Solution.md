# Part I — Foundational Concepts

## Problem 1
Construct a one-to-one correspondence between:

$\{a,b,c,d,e\}$ and 
$\{10,20,30,40,50\}$

Then explain why the actual numerical values are irrelevant.

### Solution

$a \mapsto 10,\quad \
  b \mapsto 20,\quad \
  c \mapsto 30,\quad \
  d \mapsto 40,\quad \
  e \mapsto 50$

The numerical values do not affect the correspondence 
because two sets have the same cardinality whenever 
their elements can be paired in a one-to-one correspondence.
