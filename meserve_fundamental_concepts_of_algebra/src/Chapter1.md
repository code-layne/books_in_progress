## Pre-Reading Guidance

# Important Terms

## Set
A collection of distinct objects considered as a whole.

Questions to think about:
- What makes two sets equal?
- Does order matter?
- Does repetition matter?

---

## Element
An object belonging to a set.

Notation:
\[
a \in A
\]

---

## One-to-One Correspondence
A pairing between two sets in which:
- every element in one set matches exactly one element in the other
- no elements are left unmatched

This idea becomes foundational for:
- cardinality
- functions
- equivalence
- infinite sets

---

## Cardinal Number
A measure of the “size” of a set.

Important conceptual point:
- cardinality is determined by correspondence, not magnitude

---

## Ordinal Number
Describes position/order rather than quantity.

Think carefully about:
- why ordinal and cardinal notions are logically distinct

---

## Finite Set
A set that can be matched with:
\[
\{1,2,\dots,n\}
\]

for some natural number \(n\).

---

## Infinite Set
A set for which no such finite \(n\) exists.

Watch for:
- why infinity behaves differently from ordinary arithmetic intuition

---

## Countable Set
An infinite set whose elements can still be arranged into a sequence.

Examples to analyze carefully:
- natural numbers
- integers
- rational numbers

---

## Transfinite Cardinal
A cardinality assigned to infinite sets.

Important symbol:
\[
\aleph_0
\]

You should pay careful attention to:
- why some infinite sets are “the same size”
- why some infinities are larger than others

# Important Concepts

## 1. Correspondence Defines Number

This is probably the deepest idea in the chapter.

Meserve is quietly reconstructing arithmetic from set theory.

Ask yourself:
> Is number an intrinsic property, or does it emerge from relationships?

---

## 2. Equality of Size Does NOT Mean Similar Appearance

Example:
\[
\{1,2,3\}
\]
and
\[
\{100,200,300\}
\]

have the same cardinality.

The actual values are irrelevant.

This becomes even stranger with infinite sets.

---

## 3. Infinite Sets Can Correspond to Proper Subsets

Example:
\[
n \mapsto 2n
\]

for natural numbers and even numbers.

This violates finite intuition.

Think deeply about:
- why this cannot happen for finite sets
- why it *can* happen for infinite sets

This is one of the defining properties of infinity.

---

## 4. Counting as Mapping

Children counting objects are actually:
- constructing correspondences
- building primitive functions

This connects directly to algebraic function notation later.
